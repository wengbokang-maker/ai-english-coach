# 语法批改 Prompt 模板

## 基础版：即时批改

```
Please correct my English and explain the mistakes:

My sentence: "[你的句子]"

Please:
1. Show the corrected version
2. Explain what was wrong (briefly)
3. Give me one more example of the correct usage
```

---

## 口语语法批改（不要太书面化）

```
I said this in a conversation: "[你说的话]"

Please check if there are grammar issues.
Important: I want natural spoken English, not overly formal written English.

Tell me:
- Is this grammatically correct for casual conversation?
- If not, what's the natural way to say it?
- Is this an important mistake or just minor?
```

---

## 批量批改模式

```
Please correct all these sentences I said today:

1. [句子1]
2. [句子2]
3. [句子3]
4. [句子4]
5. [句子5]

For each one: ✅ correct / ❌ needs fixing + explanation
Then at the end: tell me my most common mistake pattern today.
```
