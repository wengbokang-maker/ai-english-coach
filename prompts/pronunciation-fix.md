# 发音修正 Prompt 模板

## 快速使用版

```
I just said: "[你刚才说的句子]"

Please check if there are any pronunciation mistakes and explain:
1. Which word/sound is wrong
2. How to correct it (specific mouth position or technique)
3. A simple example sentence to practice this sound
```

---

## 针对中国英语学习者的常见问题 Prompt

```
As a pronunciation coach for Chinese English learners, help me fix these common issues:

My specific problems:
- [ ] th sound (I say "d" or "t" instead)
- [ ] final consonants (I drop them)
- [ ] word stress (I stress the wrong syllable)
- [ ] r vs l confusion
- [ ] v vs w confusion

Today's practice sentence: "[你的句子]"

Please analyze and correct.
```

---

## 日常发音检查 Prompt

```
Please give me a pronunciation check-up.
I'll say these 5 words out loud and type them:
[word1], [word2], [word3], [word4], [word5]

For each word, tell me:
- Correct phonetic spelling (IPA if possible)
- Common mistake Chinese speakers make
- One tip to remember the correct pronunciation
```
