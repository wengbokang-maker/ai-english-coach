## 🤔 这个项目是什么？

我是 **Marnie**，39 岁，商用车公司项目经理。

我正在用 AI 工具系统性地学英语——每天记录学了什么、遇到什么坑、怎么突破的。

这个仓库是我的**完整学习系统**，全程开源：
- 📋 我用的所有 AI Prompt 模板
- 🤖 自动化学习脚本（每日推送、笔记生成）
- 📝 每日学习日志（Day 1 → Day 90）
- 🗺️ 完整的 90 天学习路线图

**你可以复刻整套系统，成本接近零。**
